bool SprawdzCzyZawartoscPlikuPoprawna ( string  nazwaMapy)

Funkcja zwracająca wartość true/false czy plik jest poprawny.

Funkcja sprawdza czy zawartość pliku spełnia wymogi przewidziane przez program czyli czy są w nim same zera i jedynki, a także czy każda z tych liczb jest oddzielona spacją. Jeśli założenia pliku nie będą spełnione funkcja zwróci wartość false.

Parametry
nazwaMapyto nazwa pliku, z którego pobierana jest aktualna mapa
Zwraca
Funkcja zwraca wartość true/false w zależności od tego czy plik spełnia wymogi programu, czy też nie można z niego wyznaczyć ścieżki
