Funkcja zwracająca wartość true/false czy plik jest poprawny. 

Funkcja zwraca wartość false jeśli plik nie może zostać użyty do wyznaczenia ścieżki, to znaczy jeśli znajdują się w nim inne wartości niż przewidziane przez program czyli zera i jedynki zostanie zwrócona wartość false. Błąd zostanie również wykryty jeśli liczby w pliku nie będą oddzielone spacjami. Jeśli wszystko jest zgodne z wymaganiami programu zwrócona zostanie wartość true
